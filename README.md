# oogiri-forge 🎤

お題提案→テーマ解析→大喜利生成→GitHub公開の4エージェントパイプライン。

## クイックスタート

```bash
# お題候補を提示（番号選択でパイプライン起動）
bash suggest.sh

# お題を直接指定してパイプライン起動
bash run.sh "もし犬が人事部長だったら"

# 中断セッションを再開
bash run.sh --continue
```

## パイプライン構成

```
suggest.sh
  └─ Agent00: お題提案（Sonnet）
        ↓ 番号選択
run.sh
  ├─ Agent01: テーマ解析（Sonnet） → session.json
  ├─ Agent02: 大喜利生成（Opus）  → session.json
  └─ Agent03: 審査＆公開（Sonnet） → outputs/*.md → GitHub
```

## エージェント詳細

| # | ファイル | モデル | 役割 |
|---|---|---|---|
| 00 | agents/00_topic_suggester.md | Sonnet | 興味度×面白さでスクリーニングしたお題6本を提示 |
| 01 | agents/01_interpreter.md | Sonnet | お題を6軸で解析し解釈の引き出しを作る |
| 02 | agents/02_generator.md | Opus | 各軸2〜3本・計12〜18本の大喜利回答を生成 |
| 03 | agents/03_judge_publisher.md | Sonnet | 上位7本を審査・選別しMDファイルをGitHubに公開 |

## お題プロフィール設定

`agents/00_topic_suggester.md` のユーザープロフィールを編集することで、
生成されるお題の方向性をカスタマイズできます。

## 出力

生成された大喜利回答は `outputs/` ディレクトリにMDファイルとして保存され、
GitHubに自動プッシュされます。
